# Portfolio (Weeks 39–48)

Build 5–6 production-grade AI systems, making the architectural decisions and using AI coding tools to implement faster.

## The Mode to Work In

```
AI writes → You review → You improve → You optimize → You evaluate → You ship
```

That's already how many senior engineers work. The goal here isn't to type more code — it's to make, and be able to defend, every architectural decision in a system AI helped you build faster.

Daily loop for each project:

```
Business Problem
      ↓
Design AI Architecture
      ↓
Choose Models
      ↓
Design Retrieval
      ↓
Design Evaluation
      ↓
AI Generates Code
      ↓
Review → Debug → Optimize → Deploy → Monitor → Iterate
```

## The One Rule

Don't let AI become a crutch — if it writes all the code and you never inspect or modify it, your understanding plateaus. For every project:
- AI writes the first draft.
- You explain every important component.
- You change at least one meaningful part yourself.
- You debug at least one issue yourself.

If you can explain why every major architectural choice was made, and how the system behaves under different conditions, that's the depth expected of a senior AI engineer — even where AI wrote most of the implementation.
